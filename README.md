## 🔍 How to search for files

Use these basic steps:
1. Open fff.nvim.
2. Select the folder you want to search.
3. Type part of the file name.
4. Look at the results list.
5. Open the file you need.

You do not need the full file name. A few letters are often enough.

Examples:
- type `notes` to find files with notes in the name
- type `report` to find report files
- type `main` to find a main source file
- type `readme` to find README files

For best results, keep your search term short and specific.

To get better results:
- use short file names
- search one folder at a time
- avoid very broad terms like `a` or `file`
- use the name part you remember most
- keep your project folders clean and grouped by task

A small search term can bring back a long list, so narrow it down when needed.
